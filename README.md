# 🚀 Task Management System

A full-stack Task Management application built using React and .NET Web API with JWT Authentication.

## 🔥 Features

- 🔐 JWT Authentication & Authorization
- 📊 Dashboard with task filtering (All, Pending, InProgress, Completed, Hold)
- 🔎 Search functionality
- 📄 Pagination for task listing
- ✏️ Inline editing without page reload
- ⚡ Real-time UI updates
- ❗ API error handling & 401 session handling
- 🔄 Refresh functionality
- 🎨 Status-based color UI

## 🛠️ Tech Stack

**Frontend**
- React.js
- JavaScript
- HTML / CSS

**Backend**
- .NET Web API
- Entity Framework
- SQL Server

**Other**
- JWT Authentication
- REST API
